# Python SQLite CRUD Modular

Este repositório apresenta uma implementação robusta e modular de operações básicas de banco de dados (CRUD) utilizando **Python** e **SQLite**. 

O foco principal deste projeto é demonstrar a aplicação de boas práticas de desenvolvimento, garantindo um código limpo, seguro e fácil de manter.

### Diferenciais do Projeto
- **Context Managers (`with`)**: Garantia de fechamento seguro de conexões e cursores.
- **Row Factory**: Configuração para retorno de dados como dicionários, facilitando a leitura das colunas.
- **Tratamento de Exceções**: Gestão de erros de integridade (ex: e-mails duplicados) e falhas no banco.
- **Modularidade**: Funções isoladas e reutilizáveis para cada operação.
